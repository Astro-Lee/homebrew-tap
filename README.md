# Astro-Lee Tap

## How do I install these formulae?

`brew install astro-lee/tap/<formula>`

Or `brew tap astro-lee/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "astro-lee/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
